# HTML notes

# What is HTML
HTML (Hypertext Markup Language) is the standard markup language used to create the structure of web pages. It defines the content that appears in a browser, such as headings, paragraphs, lists, images, and links.

# HTML Tags Used
# <!DOCTYPE html>
Declares that the document is an HTML5 document

# <html>
The root element that contains the entire webpage

# <head>
Stores information about the webpage, such as metadata and the page title

# <title>
Sets the text displayed on the browser tab

# <body>
contains all the visible content shown on the webpage

# <h1>
Defines the main heading of the page

# <h2>
Defines a secondary heading used to organize sections

# <p>
Creates a paragraph of text

# <ul>
Creates an unordered list

# <li>
Represents a single item in a list

# <a>
Creates a hyperlink that allows users to navigate to another webpage

# CSS Basics
# What is CSS
CSS (Cascading Style Sheets) is a stylesheet language used to control the appearance of HTML elements. It is response for its visual design, including colors, fonts, spacing, boarders, and layout

# What is a Selector
A selector identifies which HTML element a CSS rule applies to

# Example
h1 { color : blue }

# What is a Property
A property defines what aspect of an element you want to change

# Example include
- color
- background-color
- font-size
- margin
- padding
- border

# What is a Value
A value specifies how the propert should be applied

# Example
color : blue;
Property : 'color'
Value : 'blue'
