# Deployment Notes

# What is Deployment
Deployment is the process of taking an application that works on a local computer and making it available on the internet so other people can use it

# What is Hosting
Hosting means storing a website or application on a server that is connected to the internet so users can access it through a web address

# Running Locally vs Deploying
Running locally means only the developer can access the application on their computer

Deploying means the application is available online and can be accessed by anyone with the URL

# Steps I Took to Deploy My Website 
- Created a public GitHub repository called 'my-first-webpage'
- Added my 'index.html' and 'style.css' files
- Pushed the files to GitHub
- Opened the repository settings
- Enabled GitHub Pages from the main branch
- Waited for GitHub to publish the website
- Opened the live URL in my browser
- Added the live URL to my Week 4 repository README
