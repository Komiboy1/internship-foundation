# Task 1: Study and Document Internet and Web Basics
# Internet Basics

# What is a Client: 
A client is a device or application that requests services from a server.

# What is a Server: 
A server provides resources and services to clients.

# What is DNS: 
DNS translates domain names into IP addresses.

# What is Hosting: 
Hosting stores website files on servers connected to the internet.

# What is HTTP: 
HTTP is a protocol used for communication between clients and servers.

# What is HTTPS: 
HTTPS is the secure version of HTTP using encryption.

# What is an API: 
An API allows different software systems to communicate.

# What Happens When a User Opens a Website?
The browser requests the website, DNS finds the server, the server
responds, and the browser renders the page.
