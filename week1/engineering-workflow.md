# Software Engineering Workflow

# What are Requirements
Define what needs to be built.

# What are Tasks and Tickets
Break work into manageable units.

# What is an Acceptance Criteria
Conditions that determine task completion.

# What is a Development Process
Write and implement code.

# What is a Testing Process
Verify the software works correctly.

# What is Code Review
Peers review changes before merging.

# What is Deployment
Release the application to users.

# What is Documentation
Record important project information.

# What is Feedback and Iteration
Improve based on reviews and user input.
