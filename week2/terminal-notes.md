# Terminal Notes

# What is a Terminal?
A terminal is a text-based tool that allows users to interact with their computer by typing commands instead of clicking buttons.

# pwd command
Displays the current directory or folder you are working in

# ls command
Lists all files and folders in the current directory

# cd command
Moves into the specified folder

# cd .. command
Moves one directory level up

# mkdir command
Creates a new folder

# touch file.txt
Creates a new empty file

# clear
Clears the terminal screen
