# Database Notes

# What is a Database?
A database is an oragnized system used to store, manage and retrieve data.

# Why are Database Important?
Databases allow applications to store user information, messages, orders, settings and other important data.
Without databases, information woul be lost everytime an application stopped running.

# What is a Table?
A table is a collection of related data organized into rows and columns, similar to a spreadsheet.

# What is a Row?
A row represents one complete record in a table.

# What is a Column?
A column represents a specific type of information.

# What is SQL?
SQL (Structured Query Language) is a language used to communicate with relational databases.

# Example
SElECT * FROM users; 
This query asks the database to return all users.

# Relational Databases
- MySQL
- PostgreSQL

# Characteristics
- Data stored in tables
- Uses rows and columns
- Uses SQL
- Structured relationships between data

# Non-Relational Databases
- MongoDB
- Firebase

# Characteristics
- Flexible structure
- Stores data as documents or collections
- Does not require fixed tables

# Real-World Example
I use WhatsApp every day. WhatsApp stores information such as messages, contacts, profile information, groups, photos, and videos in databases so that the data remains available whenever i open the application.
