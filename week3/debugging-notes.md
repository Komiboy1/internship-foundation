# Debugging Notes

# What is debugging
Debugging is the process of finding and fixing problems in a program. It helps developers understand why a program is not working as expected and how to correct it.

# Why is Debugging important
Debugging is important because it helps identify errors, improve software quality, and ensure that applications behave correctly

# What is an Error Message
An error message is information provided by the computer when something goes wrong in a program

# What is console.log()
console.log() is a JavaScript function that displays information in the browser's console

# Console Practice
# Practice 1
console.log(2 + 2);
Output: 4
Because both values are numbers, JavaScript performs addition

# Practice 2
console.log("2" + 2);
Output : 22
Because the first value is a string, JavaScript combines the two values into one string instead of adding them numerically
