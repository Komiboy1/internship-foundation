# Database Basics

# What is a database
A database is a collection of organized information stored electronically. It allows users and applications to store, retrieve, update, and delete data efficiently.

# What is a table
A table is a collection of related data arranged into rows and columns. In my database, the students table stores information about students

# What is a row
A row represents one complete record. for example, one student's information (name, age, course) is stored in a single row.

# What is a column
A column represents one type of information shared by all records. for example, the name column stores student names, while the age column stores student ages.

# What is a database file
A database file stores all the tables and data together. In this task, my database file is named 'practice.db'.
